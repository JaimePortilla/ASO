# Módulo 2: Gestión de Active Directory en Windows Server

Esta unidad formativa se centra en **Active Directory Domain Services (AD DS)**, la herramienta principal de Microsoft para la administración centralizada de recursos en redes Windows. A través de prácticas, se aprende a configurar y mantener un entorno de dominio seguro y eficiente.

---

## Metas de aprendizaje

Al finalizar esta unidad, serás capaz de:

- Explicar los componentes fundamentales de Active Directory (dominios, bosques, controladores de dominio).
- Instalar y promover servidores a controladores de dominio.
- Crear y gestionar cuentas de usuarios, grupos de seguridad y unidades organizativas (OU).
- Delegar permisos administrativos de forma granular.
- Diseñar y aplicar objetos de directiva de grupo (GPO) para controlar configuraciones y seguridad.
- Diagnosticar y resolver incidencias básicas en un dominio Active Directory.

La carpeta `Modulo2-ActiveDirectory` incluye los siguientes recursos:

### **Tareas Practicas**

Una serie de ejercicios guiados que cubren:

- Promoción de un servidor a controlador de dominio.
- Diseño de una estructura jerárquica con OU basada en departamentos.
- Usuarios y asignación a grupos.
- Delegación de tareas administrativas.
- Configuración de GPO para restricciones de contraseña y scripts de inicio.
- Simulación de unión de equipos al dominio y verificación.

### **Materiales de apoyo**

- Capturas de pantalla.
- Scripts PowerShell para automatización de tareas comunes.

## Temas desarrollados

En esta unidad se han abordado aspectos prácticos como:

- Implementación de AD DS y configuración inicial del dominio.
- Organización lógica del directorio mediante OU y contenedores.
- Gestión avanzada de objetos (usuarios, equipos, grupos).
- Aplicación de directivas de grupo para enforcement de seguridad.
- Uso de herramientas administrativas (Usuarios y equipos de Active Directory, Centro de administración, PowerShell).
- Identificación y corrección de problemas frecuentes, como replicación fallida o permisos incorrectos.
