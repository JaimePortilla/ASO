# UT4 Administrador remoto

En esta unidad se estudian distintas formas de administrar un servidor de manera remota, utilizando herramientas como **Windows Admin Center (WAC)** y **Cockpit**, que permiten gestionar sistemas sin necesidad de acceso físico al equipo.

Este trabajo ha sido realizado por **Jaime Portilla Pérez**, aplicando los conceptos vistos en clase para la conexión y administración remota de servidores.
